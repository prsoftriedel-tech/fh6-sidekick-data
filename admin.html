FH6 Sidekick Admin

JSON-Dateien bearbeiten, herunterladen und in dein GitHub-Pages-Repository hochladen.

Datei auswählen

Danach die heruntergeladene Datei in GitHub hochladen/ersetzen. GitHub Pages verteilt sie automatisch an die App.

Workflow

  1. Datei auswählen.
  2. Vorlage laden oder bestehendes JSON einfügen.
  3. Bearbeiten und „JSON formatieren“ klicken.
  4. Datei herunterladen.
  5. In GitHub Repository fh6-sidekick-data hochladen/ersetzen.
  6. In der App „Alle Online-Daten aktualisieren“ antippen.
